# BookLInk
## Criação de um sistema de livraria que cadastra livro e autor e associa livros a autor e autor aos seus livros
O projeto foi criado para simular um sistema de livraria que é capaz de:
- Cadastrar Livro
- Buscar Livro
  - Por editora
  - por Id
  - por titulo
  - por preço limite
- Atualizar livro
- Excluir livro

- Cadastrar autor
- Buscar autor
- Atualizar autor
- Excluir autor
